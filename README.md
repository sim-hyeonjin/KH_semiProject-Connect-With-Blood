# connect-with-blood

Description
-----------
#### 헌혈증 등록 기능을 포함한 웹사이트


#### Tech Stack

+ FE(Front-End)

   - JSP
   - CSS
   - Javascript
   - jQuery
   

+ BE(Back-End)

   - Language : Java
   - WAS(Web-Application-Server) : Apache Tomcat 8.5
   - Database
     * RDBMS : Oracle 11g XE
   - Developer tool
     * Java : Eclipse
     * Oracle : SQL-Developer

 + Design Pattern

   - MVC
